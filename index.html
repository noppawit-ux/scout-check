🔥 ตั้งค่า Firebase

เชื่อมต่อฐานข้อมูลออนไลน์เพื่อซิงค์ข้อมูลทุกอุปกรณ์

  1. เข้า firebase.google.com → สร้างโปรเจกต์
  2. ไปที่ Firestore Database → Create database → Start in test mode
  3. ไปที่ Project Settings → General → Your apps → Add app (Web icon)
  4. คัดลอก firebaseConfig มากรอกด้านล่าง
